# FinLogger Dashboard

A responsive expense tracking dashboard built with React and React Bootstrap.

## Features

- Overview cards for monthly expenses, income, and balance
- Expense summary table by category
- Expense details table with edit flow
- Add/Edit expense modal with required field validation
- Delete confirmation modal (cancel active, delete intentionally non-functional for capstone)

## Tech Stack

- React
- React Bootstrap
- Bootstrap
- Webpack

## Getting Started

### Prerequisites

- Node.js 18+
- npm 9+

### Install

```bash
npm install
```

### Run (Development)

```bash
npm start
```

The app runs on:

`http://localhost:3000`

### Build (Production)

```bash
npm run build
```
